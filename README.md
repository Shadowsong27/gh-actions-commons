# gh-actions-commons

Composite actions for CI/CD workflows across Shadowsong27 repositories.

## Available Actions

| Action | Description |
|--------|-------------|
| [`opencode-review`](./opencode-review) | Runs an OpenCode AI review on a PR diff and posts the result as a comment |

## Contributing

Each action lives in its own directory with an `action.yml` and a local `README.md` for usage details.
